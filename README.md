# Bingo lógico proposicional · Balotera Pro

Juego SCORM 1.2 listo para Brightspace enfocado en lógica proposicional, conectivos, tablas de verdad, traducción de enunciados, equivalencias, leyes de De Morgan y clasificación de fórmulas.

## Reglas principales

- La partida selecciona al azar tres figuras del cartón.
- Cada figura completada equivale a una unidad.
- Si se completan las tres unidades, el juego finaliza con nota 5.0.
- Cada respuesta correcta suma 0.2 a la nota.
- Cada respuesta incorrecta resta 0.1.
- La nota siempre se mantiene entre 0.0 y 5.0.
- El juego termina al completar las tres figuras, llegar a 5.0 o cuando el estudiante decide finalizar.
- Al terminar, se descarga automáticamente un informe HTML.
- En Brightspace, el SCORM reporta la nota sobre 5.0.

## Tipos de preguntas

- Opción múltiple de 6 opciones.
- Afirmaciones de verdadero/falso.
- Afirmaciones I, II, III y IV.
- Tablas de verdad con listas desplegables.
- Traducción de enunciados a fórmulas.
- Traducción de fórmulas a lenguaje natural.

## SCORM

El paquete reporta:

- `cmi.core.score.min = 0`
- `cmi.core.score.max = 5`
- `cmi.core.score.raw = nota sobre 5.0`
- `cmi.core.lesson_status = passed/failed`
